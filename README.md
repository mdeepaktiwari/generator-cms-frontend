# Generator CMS - Frontend

React app for generating and managing content and images. Built with Vite, React Router, and Tailwind CSS.

**Live App:** https://generator-cms-frontend.vercel.app/

## Setup

1. Install dependencies:
```bash
npm install
```

2. Make sure the backend is running and update the API base URL in `src/api.js` if needed.

3. Run the dev server:
```bash
npm run dev
```

4. Build for production:
```bash
npm run build
```

## Features

- User authentication (login/signup)
- Content generation (rewrite, expand, shorten, articles, SEO)
- Image generation with custom resolutions
- View generation history
- Protected routes

## Tech Stack

- React 19
- Vite
- React Router
- Tailwind CSS
- Axios
- React Hook Form
